Integral Domains

A zero divisor of a ring is a nonzero element \(a\) for which \(ab = 0\) for some other nonzero element \(b\). An integral domain, or just a domain, is a nonzero ring with no zero divisors.

An integral domain satisfies the cancellation law: \[ \textrm{if }ab = ac\textrm{ and } a \neq 0\textrm{, then } b = c. \]

Examples of integral domains include any subring of a field and the polynomial ring \(R[x]\) when \(R\) is an integral domain. A non-example is \(\mathbb{Z}/4\mathbb{Z}\).

Any integral domain \(R\) can be embedded in a field \(F\) called the fraction field or quotient field of \(R\) which is constructed as follows. Define a fraction to be a symbol \(\frac{a}{b}\) where \(a\) and \(b\) are in \(R\) and \(b \neq 0\). Let \(F\) be the set of equivalence classes of fractions, where \(\frac{a_1}{b_1}\) and \(\frac{a_2}{b_2}\) are equivalent if \(a_1b_2 = a_2b_1\). Sums and products are given by \(\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}\) and \(\frac{a}{b}\frac{c}{d} = \frac{ac}{bd}\). There is an injection from \(R\) to \(F\) which sends an element \(a\) to \(\frac{a}{1}\).

Furthermore, the fraction field \(F\) is the smallest field containing \(R\) in the following sense. If there is an injection from \(R\) to another field \(F'\), then there is an injection from \(F\) to \(F'\) which sends \(\frac{a}{b}\) to \(ab^{-1}\). In other words, the map \(R \to F'\) "factors through" the quotient field \(F\). It's straightforward to check that this mapping is indeed a homomorphism of rings. Its kernel is an ideal of \(F\) so it must be trivial.

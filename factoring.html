Factoring

The Fundamental Theorem of Arithmetic says that every positive integer can be written uniquely as the product of primes. We'd like to ask when something similar is true for other rings. We will introduce three classes of rings, each contained in the next: rings with a division-and-remainder algorithm (Euclidean Domains), rings in which every ideal is principal (Principal Ideal Domains), and rings with unique factorization into primes (Unique Factorization Domains).

An integral domain \(R\) is a Euclidean Domain if there is a size function \(\sigma\) such that division with remainder is possible, in the following sense. Let \(a\) and \(b\) be elements in \(R\), and suppose than \(a\) is not zero. There are elements \(q\) and \(r\) in \(R\) such that \(b = aq + r\), and either \(r = 0\) or else \(\sigma(r) < \sigma(a)\).

Some examples of Euclidean Domains:

  1. The integers \(\mathbb{Z}\) are a Euclidean domain with size function \(\sigma(a) = \vert a \vert\).

  2. A polynomial ring \(F[x]\) in one variable over a field \(F\) is a Euclidean domain with \(\sigma(f) = \textrm{degree of } f\).

  3. The Gaussian integers \(Z[i]\) are a Euclidean domain with \(\sigma(z) = \vert z \vert ^{2}\).

A greatest common divisor (or gcd) of elements \(a\) and \(b\) in a ring is a nonzero element \(d\) with

  • \(d \mid a\) and \(d \mid b\)
  • if \(d' \mid a\) and \(d' \mid b\), then \(d' \mid d\).

Given an ideal \((a, b)\) in a PID \(R\), let \(d\) be a generator such that \((d) = (a, b)\). Then

  1. \(d\) is the gcd of \(a\) and \(b\)
  2. \(d\) can be written as a linear combination of \(a\) and \(b\): there are elements \(x\) and \(y\) in \(R\) with \(d = ax + by\)
  3. \(d\) is unique up to multiplication by a unit of \(R\)
Proof: (1) Since \((a) \subset (d)\), \(a \in (d)\) and so \(d \mid a\). Similarly, \(d \mid b\). If \(d' \mid a\) and \(d' \mid b\), then \(a \in (d')\) and \(b \in (d')\) and therefore \((d) = (a,b) \subset (d')\). It follows that \(d \in (d')\) and \(d' \mid d\).

To begin discussing factorization, we start with some definitions.

  • Elements \(a\) and \(b\) are associate if they differ by a unit: there is some unit \(u\) with \(a = ub\). Equivalently, \((a) = (b)\).
  • A nonzero element \(r\) is irreducible if whenever \(r = ab\) for some \(a\) and \(b\), then at least one of \(a\) or \(b\) is a unit.
  • A nonzero element \(p\) is a prime if \(p\) is not a unit and whenever \(p\) divides a product \(ab\), then \(p \mid a\) or \(p \mid b\).

Let \(R\) be a Principal Ideal Domain.

  1. An element of \(R\) is irreducible if and only if it is a prime.
  2. The maximal ideals of \(R\) are the principal ideals generated by the irreducible elements.

Duality

I.

A linear functional is a linear map from a vector space to its field of scalars.

Some examples:

  1. \(f \colon \mathbb{R}^3 \to \mathbb{R}\) defined by \(f(x,y,z) = 4x - 5y + 2z\) is a linear functional on \(\mathbb{R}^3\).

  2. \(f \colon P_n(\mathbb{R}) \to \mathbb{R}\) defined by \(f(p) = p(1)\), the polynomial evalutated at 1, is a linear functional.

  3. Let \(V = C([0,1])\), the set of continuous functions on \([0,1]\). Define \(f \colon V \to \mathbb{R}\) as \(f(g) = \int_0^1 g(x) \, \mathrm{d}x\). Then \(f\) is a linear functional.

  4. \(f \colon \mathbb{R}^3 \to \mathbb{R}\) where \(f(x,y,z) = x^2 + y^2 + z^2\) is not a linear functional.

The dual space of a vector space \(V\), denoted \(V^*\), is the vector space of all linear functionals on \(V\). In other words, \(V^* = \mathcal{L}(V, \mathbb{F})\).

If \(V\) is finite-dimensional, then \(\textrm{dim }V = \textrm{dim }V^*\). This follows from the result that \(\textrm{dim } \mathcal{L}(V,W) = (\textrm{dim }V)(\textrm{dim }W)\).

Let \(v_1, \ldots, v_n\) be a basis for \(V\). The dual basis of \(v_1, \ldots, v_n\) is the list \(f_1, \ldots, f_n\) of elements of \(V^*\), where \(f_j(v_k) = \delta_{kj}\). Since linear maps are completely determined by their actions on the basis vectors, this is enough to define \(f_j\).

The dual basis of a basis of \(V\) is a basis for \(V^*\).

Proof: Suppose \(v_1, \ldots, v_n\) is a basis for \(V\). Let \(f_1, \ldots, f_n\) denote the dual basis.
To show that \(f_1, \ldots, f_n\) are linearly independent, suppose \(a_1f_1 + \cdots + a_nf_n = 0\), the zero functional. In particular, \[ (a_1f_1 + \cdots + a_nf_n)(v_1) = a_1f_1(v_1) + \cdots + a_nf_n(v_1) = a_1 = 0. \] Similarly, \(a_2 = \cdots = a_n = 0\). Hence \(f_1, \ldots, f_n\) are linearly independent.
Since \(\textrm{dim }V^* = \textrm{dim }V\), the dual basis spans \(V^*\).

To show explicitly how an arbitrary \(f \in V^*\) can be written as a linear combination of the dual basis \(f_1, \ldots, f_n\), we need to find \(a_1, \ldots, a_n\) so that \[ f(x) = a_1f_1(x) + \cdots + a_nf_n(x) \textrm{ for all } x \in V. \] We must have \[ f(v_j) = a_1f_1(v_j) + \cdots + a_nf_n(v_j) = a_j, \] so \(a_j = f(v_j)\). To show that these \(a_1, \ldots, a_n\) work, note that \[ \begin{aligned} f(v_1)f_1(x) + \cdots + f(v_n)f_n(x) &= x_1f(v_1) + \cdots + x_nf(v_n) \\ &= f(x_1v_1 + \cdots + x_nv_n) \\ &= f(x). \end{aligned} \]

Example. Let \(V = \mathbb{R}^2\) and \(\beta = \{(2,1),(3,1)\}\) be a basis for \(V\). We want to find the dual basis \(\beta^* = \{f_1, f_2\}\).
\(\,\,\,\,\) We have \(f_1(2,1) = 1\) and \(f_1(3,1) = 0\), which gives the system \[ \begin{aligned} 2f_1(1,0) + f_1(0,1) &= 1 \\ 3f_1(1,0) + f_1(0,1) &= 0. \end{aligned} \] Solving the system yields \[ f_1(1,0) = -1, \,\,\,\, f_1(0,1) = 0.\] So \[ f_1(x,y) = xf_1(1,0) + yf_1(0,1) = -x + 3y.\] Similarly, \[ f_2(x,y) = x - 2y.\] \(\,\,\,\,\) Using the formula above, we can write any \(f \in V^*\) as \[ f = f(2,1)f + f(3,1)f .\] For example, if \(f(x,y) = 2x - 5y\), then \(f\) can be written \[ f = -f_1 + f_2 = -(-x + 3y) + (x - 2y). \]

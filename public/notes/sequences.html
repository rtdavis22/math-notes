Sequences

I.

A sequence of real numbers \(a_n\) is a Cauchy sequence if for every \(\varepsilon > 0\), there is an \(N\) such that \(\vert a_n - a_{n'} \vert \leq \varepsilon\) for all \(n, n' \geq N\).

A sequence \(a_n\) converges to a real number \(L\) if for every \(\varepsilon > 0\), there is an \(N\) such that \(\vert a_n - L \vert \leq \varepsilon\) for every \(n \geq N\). It is not possible for a sequence to converge to two distinct real numbers. If \(a_n\) converges to \(L\), we say that the limit of \(a_n\) is \(L\) and write \(L = \lim_{n \to \infty} a_n\).

Convergent sequences are Cauchy.

A sequence \(a_n\) is bounded by a real number \(M\) if \(\vert a_n \vert < M\) for all \(n\). We say that a sequence is bounded if it is bounded by some \(M > 0\).

Convergent sequences are bounded.

Let \(x = \lim_{n \to \infty} a_n\) and \(y = \lim_{n \to \infty} b_n\). We have the following limit laws:

  1. \(\lim_{n \to \infty} (a_n + b_n) = x + y\).
  2. \(\lim_{n \to \infty} (a_nb_n) = xy\).
  3. For any real number \(c\), \(\lim_{n \to \infty} (ca_n) = cx\).
  4. If \(y \neq 0\) and \(b_n \neq 0\), then \(\lim_{n \to \infty} b_n^{-1} = y^{-1}\).
  5. If \(y \neq 0\) and \(b_n \neq 0\), then \(lim_{n \to \infty} \frac{a_n}{b_n} = \frac{x}{y}\).
  6. \(\lim_{n \to \infty} \max(a_n, b_n) = \max(x, y)\).
  7. \(\lim_{n \to \infty} \min(a_n, b_n) = \min(x, y)\).

An upper bound \(M\) of a set of real numbers \(E\) is called the supremum of \(E\) if any other upper bound \(M'\) of \(E\) is larger than \(M\). The supremum of a set \(E\) is denoted \(\sup(E)\). Similarly, the infimum of a set \(E\) is the greatest lower bound of \(E\) and can be defined by the formula \(\inf(E) = -\sup(-E)\). The supremum and infimum of a sequence \(a_n\) are the supremum and infimum of the set \(\{a_n\}\).

Monotone bounded sequences converge.

II.

A real number \(x\) is a limit point of a sequence \(a_n\) if for every \(\varepsilon > 0\) and integer \(N\), there is an \(n \geq N\) such that \(\vert a_n - x \vert \leq \varepsilon\). For example, if the limit of a sequence exists, then it is a limit point.

The limit superior of a sequence \(a_n\) is defined \[ \limsup\limits_{n \to \infty} a_n = \lim\limits_{n \to \infty} (\sup\limits_{m \geq n} a_m). \] Note that the sequence on the right is a decreasing sequence. Similarly, the limit inferior is defined \[ \liminf\limits_{n \to \infty} a_n = \lim\limits_{n \to \infty} (\inf\limits_{m \geq n} a_m). \] For example, consider the sequence \[ 1.1, -1.01, 1.001, -1.0001, 1.00001, \ldots\] The limit superior of this sequence is 1 and the limit inferior is -1.

Let \(a_n\) be a sequence with limit superior \(L^+\) and limit inferior \(L^-\). We have the following properties:

  1. For every \(x > L^+\), there is an \(N\) such that \(a_n < x\) for all \(n \geq N\). Similarly, for every \(x < L^-\), there is an \(N\) such that \(a_n > x\) for all \(n \geq N\).
  2. For every \(x < L^+\) and every \(N\), there is an \(n \geq N\) such that \(a_n > x\). Similarly, for every \(x > L^-\) and every \(N\), there is an \(n \geq N\) such that \(a_n < x\).
  3. \(\inf a_n \leq L^- \leq L^+ \leq \sup a_n\).
  4. If \(c\) is any limit point of \(a_n\), then \(L^- \leq c \leq L+\).
  5. If \(L^+\) is finite, then it is a limit point of \(a_n\). Similarly, if \(L^-\) is finite, then it is a limit point of \(a_n\).
  6. If \(a_n\) converges to \(c\), then \(L^+ = L^- = c\). Conversely, if \(L^+ = L^- = c\), then \(a_n\) converges to \(c\).

The comparison principle says that if \(a_n \leq b_n\), then we have the following:

  1. \(\sup a_n \leq \sup b_n\)
  2. \(\inf a_n \leq \inf b_n\)
  3. \(\limsup_{n \to \infty} a_n \leq \limsup_{n \to \infty} b_n\)
  4. \(\liminf_{n \to \infty} a_n \leq \liminf_{n \to \infty} b_n\)

The squeeze test follows: if \(a_n \leq b_n \leq c_n\) and \(a_n\) and \(c_n\) both converge to \(L\), then \(b_n\) also converges to \(L\). And from this follows the zero test: the limit \(\lim_{n \to \infty} a_n\) exists and is equal to zero if and only if \(\lim_{n \to \infty} \vert a_n \vert\) exists and is equal to zero.

A sequence \(a_n\) is convergent if and only if it is a Cauchy sequence.

III.

Let \(a_n\) and \(b_n\) be sequences. We say that \(b_n\) is a subsequence of \(a_n\) if there is a function \(f \colon \mathbb{N} \to \mathbb{N}\) which is strictly increasing such that \(b_n = a_{f(n)}\) for all \(n \in \mathbb{N}\).

A sequence \(a_n\) converges to \(L\) if and only if every subsequence of \(a_n\) converges to \(L\).

A real number \(L\) is a limit point of \(a_n\) if and only if there exists a subsequence of \(a_n\) which converges to \(L\).

(Bolzano-Weierstrauss Theorem). Every bounded sequence has at least one convergent subsequence.
